What The Heck Is Responsive Web Design?

Responsive websites “respond” to their environment (primarily based on the width of the browser window)

Adaptive vs. Responsive

Multiple Fixed Width Designs vs. Fluid Grid System

The Goldilocks Approach

Fixed width for large and medium, fluid for small.

Why?

Create websites that work across all screen sizes and devices.
